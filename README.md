# Sistema de Gestión de Ventas

## Descripción

Sistema web desarrollado con Node.js, Express, MySQL, Sequelize y EJS para la gestión de productos en una tienda.

El proyecto permite:

- Agregar productos
- Editar productos
- Eliminar productos
- Visualizar productos
- Conectar con MySQL
- Utilizar una API REST
- Frontend dinámico con EJS

## Tecnologías

- Node.js
- Express.js
- MySQL
- Sequelize
- EJS
- GitHub
- Render
- Aiven
